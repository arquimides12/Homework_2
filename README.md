# Real-Time Task Progress App
## Description (English)

This project demonstrates the use of Sockets and concurrency in a Node.js environment.
It allows users to visualize the progress of a task in real time through a simple and responsive web interface.
The system uses Express for the server, Socket.IO for real-time communication, and Winston for structured event logging.

## Main Features

Real-time updates using Socket.IO.

Web interface that displays task progress dynamically.

Log management with Winston, recording connection and event messages.

Concurrent handling of multiple client connections without blocking the system.

## Technologies Used

- Node.js
- Express
- Socket.IO
- Winston
- Nodemon (for automatic server restart during development)
